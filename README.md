# Парсер вакансий hh.ru

Скрипт собирает данные о вакансиях с сайта hh.ru по заданным критериям и сохраняет их в базу данных PostgreSQL.

## Возможности
- Поиск по нескольким ключевым словам.
- Настройка региона, периода, количества страниц.
- Сбор ключевых навыков (опционально).
- Сохранение результатов в CSV или PostgreSQL.
- Поддержка прокси для обхода ограничений.
