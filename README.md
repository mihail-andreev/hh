# Парсер и аналитик вакансий hh.ru 🎯

Комплексное решение для сбора, хранения и экспорта данных о вакансиях с сайта hh.ru. Проект включает в себя парсер вакансий, базу данных PostgreSQL и инструменты для экспорта данных в Excel.

## 📋 Возможности

### Парсер вакансий (`api_hh.py`)
- 🔍 Поиск по нескольким ключевым словам
- 📍 Настройка региона и периода публикации
- 📄 Сбор ключевых навыков из вакансий
- 🔄 Автоматическая смена прокси при блокировках
- 🛡️ Система повторных попыток при ошибках
- 📊 Сохранение в базу данных PostgreSQL

### Экспорт в Excel (`sql_to_excel.py`)
- 📤 Выгрузка данных из БД в Excel
- 🔄 Инкрементальное обновление (только новые вакансии)
- 🚫 Защита от дублирования записей
- 📁 Автоматическое создание папки для экспорта
- 🏷️ Информативные сообщения о результате

## 🛠️ Технологии

- **Python 3.9+**
- **PostgreSQL** - хранение данных
- **Pandas** - обработка и экспорт данных
- **Requests** - HTTP запросы к API hh.ru
- **BeautifulSoup4** - парсинг списка прокси
- **psycopg2** - работа с PostgreSQL
- **openpyxl** - работа с Excel файлами

## 📦 Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/mihail-andreev/hh.git
   cd hh
   ```
2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # для Linux/macOS
   # или
   venv\Scripts\activate     # для Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте базу данных PostgreSQL:**
   ```bash
   CREATE DATABASE hh_api_data;
   ```
5. **Создайте файл конфигурации config.py: (используйте шаблон ниже)**

## Файл config.py
   ```bash
   DB_CONFIG = {
      'host': 'localhost',
      'port': '5432',
      'database': 'hh_api_data',
      'user': 'postgres',
      'password': 'your_password'
   }
   ```
